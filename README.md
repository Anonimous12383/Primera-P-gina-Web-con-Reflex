# 🌐 Mi Primera Página Web con Reflex

## 📖 Descripción del Proyecto

Este proyecto consiste en la creación de una primera página web utilizando **Reflex**, un framework moderno que permite desarrollar aplicaciones web full stack utilizando únicamente Python.

El objetivo principal fue aprender el proceso completo de configuración de un entorno de desarrollo web con Reflex, incluyendo:

- Creación de un repositorio en GitHub.
- Configuración del proyecto con Poetry.
- Instalación y configuración de Reflex.
- Creación y ejecución de una aplicación web básica.
- Diseño de una interfaz sencilla con componentes dinámicos.
- Implementación de un contador interactivo.

La aplicación desarrollada incluye una página principal personalizada con un contador funcional que incrementa y disminuye valores mediante botones interactivos.
